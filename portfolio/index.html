Telemetry Application Oregon State Solar Vehicle Team

This application is designed to make it easy for team members in the pit to see remote data from the solar car.

  • Application built using Flask Framework and SQLAlchemy
  • Displays live data from MySQL database
  • Interface styled using the Twitter Bootstrap
  • Google Map API used to render position of Car and Update the Map with it's path
  • Graphs are rendered using the HighCharts API
  • Ajax is used to fetch data encoded in json from the server
  • Asynchronous loading allows the display of a large amount of data, currently tested up to 300,000 data points
